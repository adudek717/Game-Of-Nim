# This is a Game of Nim implementation using Python and easyAI

Short instruction:
Run the game:
    - python3 nim.py

Select a row and the number of matches (0 or 1) in
the following format: "row-matches"
For example:
    - 0-1
    - 3-2
    - 1-1
    - 1-2
